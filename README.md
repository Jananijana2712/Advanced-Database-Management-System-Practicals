# Advanced Database Management System (ADBMS) Practicals

## Overview

This repository contains the laboratory practicals completed as part of the **Advanced Database Management System (ADBMS)** course during my M.Tech in Information Technology.

The practicals cover various modern database technologies, including NoSQL databases, Temporal Databases, and Graph Databases, providing hands-on experience with different database models and query languages.

## Practicals Covered

### MongoDB

* Database and Collection Creation
* CRUD Operations
* Embedded Documents
* Aggregation Framework
* Indexing
* Query Operations
* Document-Based Data Modeling

### Apache Cassandra

* Keyspace Creation
* Table Creation
* CQL (Cassandra Query Language)
* CRUD Operations
* Partition Keys
* Data Modeling
* Distributed Database Concepts

### Temporal Database

* Temporal Data Concepts
* Valid Time
* Transaction Time
* Historical Data Management
* Temporal Queries
