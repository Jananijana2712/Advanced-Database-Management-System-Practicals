# Advanced Database Management System (ADBMS) Practicals

## Overview

This repository contains the laboratory practicals completed as part of the **Advanced Database Management System (ADBMS)** course during my M.Tech in Information Technology.

The practicals provide hands-on experience with modern database technologies, including NoSQL databases, Graph Databases, XML Databases, Multimedia Databases, and Temporal Databases.

## Practicals Covered

### MongoDB

* Database and Collection Creation
* CRUD Operations
* Embedded Documents
* Aggregation Framework
* Indexing
* Query Operations
* Document-Based Data Modeling

### Apache Cassandra

* Keyspace Creation
* Table Creation
* CQL (Cassandra Query Language)
* CRUD Operations
* Partition Keys
* Data Modeling
* Distributed Database Concepts

### XML Database

* XML Document Creation
* XML Schema (XSD)
* DTD (Document Type Definition)
* XPath Queries
* XQuery

### Multimedia Database

* Multimedia Metadata

* Content-Based Multimedia Queries

### Temporal Database

* Temporal Data Concepts

* Historical Data Management
